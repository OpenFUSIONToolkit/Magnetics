Grounded, quote-backed summaries of the magnetics resources (in `../../resources/`), one per
document. Produced by reading each source in full — focused on hardware, the SLCONTOUR
(quasi-stationary) and MODESPEC (rotating-mode) tools, analysis methods, the visualizations to
reproduce, and concrete example values. See `../VISION.md` for the synthesis.
